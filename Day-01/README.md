# Day 01 – Setup & Tooling

## Learning Objectives

- Install and configure JDK 17+ for Java development
- Set up Node.js and npm for frontend development
- Configure Git and connect to GitHub
- Choose and set up an IDE (IntelliJ IDEA / VS Code)
- Verify all tools work together with a simple test

## Topics Covered

- JDK installation and `JAVA_HOME` configuration
- Node.js and npm setup, `node -v` / `npm -v` verification
- Git installation, `git config` (username, email)
- GitHub account setup and SSH key configuration
- IDE installation and essential extensions/plugins
- Terminal/CLI basics for development workflow

## Hands-On Task

Set up your complete development environment:
1. Install JDK 17 and verify with `java --version` and `javac --version`
2. Install Node.js (LTS) and verify with `node -v` and `npm -v`
3. Configure Git with your name and email
4. Generate an SSH key and add it to your GitHub account
5. Clone a test repository to verify everything works
6. Open the project in your IDE and run a "Hello World" in both Java and JavaScript

## Practice / Homework

Create a GitHub repository for this internship. Add a README.md describing your goals for the next 30 days. Make your first commit and push it.
